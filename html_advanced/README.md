# 🚀 HTML Advanced Project

## 📋 Project Overview

This project focuses on learning HTML5 tags to structure a web page without CSS or styling. The goal is to understand the semantic structure of HTML.

## 🎯 Key Learning Objectives

- 📏 Follow HTML guidelines
- 🏗️ Create the skeleton of an HTML5 page
- 🏷️ Use semantic HTML tags for page structure
- 🧠 Understand when to use div vs span
- 🔍 Recognize the semantic value of header, main, footer, article, nav, section, aside
- 📊 Use headings properly and understand their hierarchical importance
- 📝 Create lists in HTML
- 🖼️ Differentiate between media types (SVG, GIF, PNG, JPG)
- 📊 Structure data in tables
- 🎥 Integrate video and audio in a webpage
- 🔗 Embed external content
- 🏗️ Correctly structure an HTML page

## 📌 Project Requirements

- 📄 Include a README.md file at the root of the project folder
- ✅ Ensure code is W3C compliant and validates with W3C-Validator
- 🏢 Use "Techium" as the company name across webpages

## 📝 Key Tasks

1. 🔰 Create basic HTML structure with doctype, html, head, and body tags
2. 🔤 Add meta tags for charset and viewport
3. 📌 Include title, description, and favicons
4. 🏗️ Create header, main, and footer sections
5. 📚 Add semantic sections within the main content
6. 📰 Include articles within specific sections
7. 🧭 Create navigation within the header
8. 📊 Add headings of different levels (h1-h6) in appropriate sections
9. 📝 Include paragraphs in various sections
10. 🔳 Use span and div elements for content grouping
11. 🏗️ Structure sections with header and div elements

## 📝 Additional Notes

- 🔍 Pay attention to details like lowercase vs uppercase
- 🗺️ Follow the provided sitemap and wireframe for the Techium project
- 📚 Create a styleguide page to showcase different HTML elements

This project aims to build a strong foundation in HTML structure and semantics, which is crucial for creating well-organized and accessible web pages. 🌟
